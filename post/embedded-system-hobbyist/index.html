Embedded System Hobbyist

August 18, 2021

Over about 10 years of being embedded system hobbyist I’d worked with various evaluation boards.
Here is short story of me and my evaluation boards!

Raspberrypi Model B

It was around 1391 when I purchased this board. My first Linux board
I

  • Learn about cross compile and mounting rootfs over NFS
  • Read Embedded Linux Primer by Christopher Hallinan
  • Read Embedded Linux training by bootlin

Tiny6410

In summer of 1393 I purchased this board from ECA eshop.
I learn how to use openocd to initialize CPU and it’s peripherals
NOTE: It’s Not a very good evaluation board for learning purpose

F746-DISCO

I purchased this board around 1394 to learn about

  • cortex-m CPUs (The Designer’s Guide to the Cortex-M Processor Family by Trevor Martin)
  • STM32 (Mastering STM32 by Carmine Noviello)
  • Mbed framework

Nanopi

In summer of 1394 I purchased this from Aftabrayaneh
I was trying to build 2G modem.
I learn about ppp over serial port (Internet access via SIM800C module)
But it’s internal WiFi module (AP6210) didn’t WORK!

Nanopi-Neo

Around 1395 I purchased this board which was a total mistak!

Sam9-eval-v1

I always had a dream of building my own evluation board.
I did this on 1396 and built my very own linux board based on ATSAM9260 SOC (SAM9-EVAL)
It’s Ethernet didn’t work at first place (never foundout why) It’s SDRAM had some signal/power integrity issues

Z-turn

In about 1396, I was totaly obsessed with Zynq-7000 family So I purchased this board and

  • Learn about Xilinx Zynq-7000 family (Thanks to Dr.sadri)
  • Cross compile U-boot/Kernel/ROOTFS for Z-turn
  • Create simple AXI-MM peripheral and struggle with cache coherency

Sam9-eval-v2

At the end of 1396 I fixed issues of Sam9-eval-v1 which leads to Sam9-eval-v2
I

  • Fixed Ethernet and signal integrity issues in V1
  • Patched at91bootstrap according to designed HW
  • Cross compiled at91bootstrap/u-boot/kernel/rootfs
  • Learn alot about buildroot

VS-MP3

I guess 1396 was very good year for me!
I also designed my very own MP3 player based on STM32F103RC/VS1360
It’s development contiuses till 1400
I

  • Learn about FreeRTOS (Hands-On RTOS with Microcontrollers by Brian Amos)
  • Learn about STemWin / LVGL

EPCI

So I’m back again in 1398 with brilliant idea!
Let’s re-invent the wheel!
I don’t know why, but I need to understand things down to the wires.
So I designed EPCI which I wanted to be an open source NIC.
I

  • Learn about PCI bus (PCI Bus Demystified By Doug Abbott)
  • Learn about Linux device driver (Linux Device Drivers, by Jonathan Corbet, Alessandro Rubini, and Greg Kroah-Hartman)

Asus Tinker

Back again in 1399, I wanted to learn about Android and embedded Android.
So I tryied to compile AOSP form source and getting hands on embedded Android (Embedded Android by Karim Yaghmour)

Omega2+

Never endding obsession!
In 1400 I want to learn about OpenWRT
I also belive that Omega2+ is most adorable Linux board.
